## Agent skills

### Issue tracker

本仓库使用 GitHub Issues 作为 issue 跟踪器。详细说明见 docs/agents/issue-tracker.md。

### Triage labels

使用标准的 triage 标签词汇：`needs-triage`、`needs-info`、`ready-for-agent`、`ready-for-human`、`wontfix`。详细映射见 docs/agents/triage-labels.md。

### Domain docs

本仓库为单一上下文（single-context）：根目录包含 `CONTEXT.md`，架构决策放在 `docs/adr/`。更多规则见 docs/agents/domain.md。
